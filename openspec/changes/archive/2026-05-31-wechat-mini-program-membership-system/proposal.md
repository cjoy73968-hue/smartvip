## Why

微信小程序会员管理系统帮助商家标准化管理会员、自动化运营活动、激励会员裂变拉新。系统支持会员入驻、余额充值消费、推广返利、自动派单、周期礼品派送等全流程功能，适配实体门店、本地生活、私域电商等多类经营场景。

## What Changes

- **会员入驻模块**：双入口（小程序码/公众号）入驻，微信授权登录，手机号绑定，地址管理
- **会员订阅机制**：会员购买/续费，有效期管理，自动订阅，到期提醒
- **账户体系**：双账户结构（充值余额/返利余额），消费优先扣充值余额
- **余额充值消费**：微信支付充值，余额消费，账单明细
- **推荐裂变返利**：专属推荐码，上下级绑定，返利比例配置，7天到账机制
- **周期礼品派送**：商家配置默认礼品和派送周期，用户选择礼品按差价扣费
- **一次性派送**：用户随时下单，全额支付
- **商家后台**：会员管理、订单管理、商品管理、返利配置、配送配置、数据报表、营销推送
- **消息通知**：微信模板消息推送（派送通知、余额不足、返利到账、到期提醒等）

## Capabilities

### New Capabilities

- **member-onboarding**: 会员入驻流程，支持小程序码和公众号双入口，微信授权，手机绑定，地址管理，推荐关系绑定
- **membership-subscription**: 会员订阅机制，会员购买/续费，有效期管理，自动订阅，到期提醒，权益停止/恢复
- **dual-account-system**: 双账户体系，充值余额和返利余额独立管理，消费优先规则，余额安全
- **recharge-consumption**: 余额充值与消费，微信支付对接，实时到账，账单记录
- **referral-rebate**: 推荐裂变返利，推荐码生成，上下级绑定，返利计算，7天到账，返利追回
- **gift-delivery**: 礼品派送模块，周期自动派送（差价模式）和一次性派送（全额模式），配送范围，状态管理
- **merchant-backend**: 商家PC后台，会员管理、订单派送、商品管理、返利配置、会员配置、配送配置、数据报表、营销推送
- **wechat-notifications**: 微信服务通知，模板消息推送
- **auto-subscribe**: 自动订阅功能，到期前自动扣款续费

### Modified Capabilities

- （无 - 新项目无现有规格）

## Impact

- **前端**：微信小程序用户端（登录、会员、商品、订单、礼品派送），Vue3+Element Plus商家后台
- **后端**：Java 17 + Spring Boot 3 + MyBatis-Plus + MySQL 8.0 + Redis 7
- **第三方**：微信支付API、微信模板消息API、七牛云OSS
- **部署**：腾讯云轻量应用服务器 + Docker
