# 小程序发布指南

## 一、配置小程序信息

编辑项目根目录下的 `project.config.json`，填写你的小程序 appid：

```json
{
  "appid": "wx1234567890abcdef"
}
```

> **获取 AppID**：登录 [微信公众平台](https://mp.weixin.qq.com) → 开发管理 → 开发设置 → AppID

---

## 二、在微信开发者工具中导入项目

1. 打开**微信开发者工具**
2. 点击**导入项目**
3. 选择项目目录：`/data/smartvip/miniprogram/`
4. 填写 AppID（与上方配置一致）
5. 点击**确定**

---

## 三、配置服务器域名

在微信开发者工具中操作：

1. 点击右上角**详情**
2. 选择**项目设置**
3. 在**域名信息**中添加合法域名：

| 域名类型 | 填写内容 |
|---------|---------|
| request合法域名 | `http://你的服务器IP:8080` |
| uploadFile合法域名 | 你的七牛云存储域名（可选） |
| downloadFile合法域名 | 你的七牛云存储域名（可选） |

> 如果没有配置七牛云，可以暂时留空

---

## 四、修改小程序接口地址

编辑 `/data/smartvip/miniprogram/api/api.js`，将 `baseUrl` 修改为测试服务器地址：

```javascript
const baseUrl = 'http://你的服务器IP:8080'
```

---

## 五、上传体验版

1. 在微信开发者工具中点击**上传**
2. 填写版本号（如 `1.0.0`）和备注
3. 登录 [微信公众平台](https://mp.weixin.qq.com)
4. 进入**管理** → **版本管理**
5. 找到刚上传的版本，点击**体验版**

---

## 六、测试人员邀请

1. 在微信公众平台 → **管理** → **成员管理**
2. 添加测试微信号
3. 测试人员即可在微信中访问体验版小程序

---

## 注意事项

1. **域名必须备案**：request 合法域名必须已备案
2. **HTTPS**：生产环境必须使用 HTTPS
3. **版本号**：每次上传需要递增版本号
4. **体验版有效期**：体验版有效期为 30 天
